Алгоритм Пирсона - это простой алгоритм хэширования, который использует операции сложения и умножения для создания хэш-значения строки.

Вот как работает этот алгоритм:

1. Начнем с нулевого значения хэша.
2. Для каждого символа в строке:
	- Получим числовое значение символа (его код Unicode, если мы работаем с ASCII-символами).
	- Умножим текущее значение хэша на некоторое простое число (обычно выбирают 31).
	- Прибавим к этому результату числовое значение символа.
	- Полученное значение будет новым значением хэша.
3. Повторяем этот процесс для каждого символа в строке.
4. В конце получаем хэш-значение всей строки.

Преимущество алгоритма Пирсона заключается в его простоте и относительной эффективности.
Он хорошо подходит для быстрого вычисления хэш-значений для текстовых данных. Однако стоит отметить, что он не является
криптографически стойким и не подходит для использования в криптографических целях.

Рассмотрим пример работы алгоритма Пирсона на строке "hello":

1. Начальное значение хэша: 0.
2. Для каждого символа в строке:
	- Символ 'h', числовое значение: 104.
		Новое значение хэша: 0 * 31 + 104 = 104.
	- Символ 'e', числовое значение: 101.
		Новое значение хэша: 104 * 31 + 101 = 3225.
	- Символ 'l', числовое значение: 108.
		Новое значение хэша: 3225 * 31 + 108 = 100173.
	- Символ 'l', числовое значение: 108.
		Новое значение хэша: 100173 * 31 + 108 = 3100911.
	- Символ 'o', числовое значение: 111.
		Новое значение хэша: 3100911 * 31 + 111 = 96113152.
3. Таким образом, хэш-значение для строки "hello" с использованием алгоритма Пирсона равно 96113152.
